# TikTok Reel Clone

Questo progetto è una copia di un reel TikTok realizzata **solo a scopo didattico**.  
È stato creato per fare pratica con HTML e CSS.

## Disclaimer

Questo progetto non è affiliato a TikTok.  
Tutti i diritti del design e dei contenuti originali appartengono ai rispettivi proprietari.  
Il progetto è usato solo per studio.

## Descrizione

Il progetto riproduce l’aspetto grafico di un reel TikTok.  
Non ci sono funzioni interattive e non viene usato JavaScript.

Serve solo per:
- fare pratica con HTML
- fare pratica con CSS
- imparare a organizzare file e cartelle

## Tecnologie usate

- HTML
- CSS

## Struttura del progetto

index.html  
css/  
 └─ style.css  
assets/  
 ├─ img/  
 └─ video/  

## Come aprire il progetto

1. Scarica o clona il repository
2. Apri il file `index.html` con un browser (Chrome, Edge, Firefox, ecc.)

Non serve installare nulla.

## Obiettivo

Imparare a:
- creare una pagina web
- usare CSS per il layout
- gestire immagini e video
